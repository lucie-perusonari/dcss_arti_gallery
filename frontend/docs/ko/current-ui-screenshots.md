# 현재 UI 스크린샷

`frontend/reference/screenshots/current-ui/`는 Gallery API 데이터로 확인한 현재 React 화면 흐름입니다.
실제 DCSS 디자인 reference가 아니며, 구현 변경 전후 회귀 비교용으로만 사용합니다.

- `01-gallery-default.png`: full gallery
- `02-filter-jewellery.png`: jewellery filter state
- `03-weapon-detail.png`: weapon detail state
- `04-mobile-search.png`: mobile search state
- `mobile-modal-check.png`: mobile item detail modal state
