# `src/types/status.ts`

`src/types/status.ts`는 crawl operations dashboard가 소비하는 TypeScript 타입을 소유합니다.

## 책임

- `CrawlStatus`, `RawFileStatus`, `LatestActivity`, `CrawlError`, `GalleryApiMetrics` 타입을 정의합니다.
- Admin API response DTO와 frontend 사용 경계를 정렬합니다.

## 변경 규칙

타입을 변경하면 [Admin API Data Types](../../../admin_api/docs/ko/data-types.md)를 함께 확인합니다.
