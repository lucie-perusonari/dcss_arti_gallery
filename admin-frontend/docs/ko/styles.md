# `src/styles.css`

`src/styles.css`는 crawl operations dashboard의 layout과 상태 표시 스타일을 소유합니다.

## 책임

- dashboard grid, card, table, status badge, error list의 시각 표현을 정의합니다.
- 운영 상태를 빠르게 스캔할 수 있는 밀도와 대비를 유지합니다.
- component 구조 변경 없이 가능한 presentation 조정을 담당합니다.
